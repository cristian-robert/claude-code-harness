# inbox / research

**Deep-dive briefs.** Gathered sources, comparisons, and findings on a topic — before they're distilled into an evergreen `wiki/` or `agent-kb/` page. One note (or subfolder) per topic.

## Contents

_Empty. One note (or subfolder) per topic — filename `<topic>.md`, or `<topic>/` with its own `_index.md` for larger investigations._

## Agent SOP

1. New investigation → `<topic>.md`. If it grows past one file, promote it to a subfolder `<topic>/` **and give that subfolder an `_index.md`** (Index Law).
2. Cite sources as standard Markdown links; link related vault notes with `[[wikilinks]]`.
3. When conclusions stabilize, harvest them into `wiki/` or `agent-kb/` and leave a pointer here.
4. Update this index when you add/move a brief.
